Keyboard Shortcuts

Navigation

Navigate up and down through the Concepts
Expand a Concept to show its children (or click on the blue arrow icon with your mouse).
Collapse a Concept to hide its children

Concept Edits

ENTER
Create a new Concept
DELETE
Delete current Concept
CTRL+
Move Concept up
CTRL+
Move Concept down
CTRL+
Move Concept left (outdent)
CTRL+
Move Concept right (indent)

TAB
Move Concept right (indent)
SHIFT+TAB
Move Concept left (outdent)

Phrasing and Document

ESCAPE
Edit phrasing
SHIFT+ENTER
Create new phrasing (and select it)
SHIFT+
Change selected phrasing (for the current document) to previous phrasing in list
SHIFT+
Change selected phrasing (for the current document) to next phrasing in list
BACKSPACE
Delete phrasing
CTRL+ALT+D
Create new Document
SHIFT+LEFT
Change active Document one tab to the left
SHIFT+RIGHT
Change active Document one tab to the right

CTRL+U
Toggle unordered list (i.e. bullets) on current Concept
CTRL+O
Toggle ordered list (i.e. numbers) on current Concept
CTRL+H
Toggle current Concept as heading
CTRL+I
Toggle image view (thumbnail, normal, icon) for entire project

CTRL+ALT+C
Add a new comment on the current Concept and in the current Document

General

CTRL+P
Print current Document
CTRL+ALT+X
Open/Close Properties View (PVs)
CTRL+ALT+N
Create new Project
CTRL+ALT+I
Import Project
CTRL+K
Show Keyboard Shortcuts
